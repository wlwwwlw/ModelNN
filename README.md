# ModelNN

Веб-приложение для построения полицентрической модели сельской агломерации на данных Нижегородской области.

Приложение разработано в рамках выпускной квалификационной работы по теме:

**«Анализ мотивации проживания на сельских территориях методами машинного обучения»**

## Описание проекта

Данный репозиторий содержит исследовательскую версию приложения, в которой выполняется сравнение нескольких моделей кластеризации для анализа сельских территорий Нижегородской области.

Приложение позволяет загрузить данные о:

* численности населения территорий;
* объектах инфраструктуры;
* дорожных расстояниях между территориями.

На основе этих данных рассчитываются показатели обеспеченности и качества инфраструктуры, после чего формируется матрица полицентрической связи между территориями.

## Используемые методы

В приложении реализован подбор и сравнение нескольких моделей кластеризации:

* KMeans;
* Agglomerative Clustering;
* DBSCAN;
* OPTICS;
* Gaussian Mixture Model;
* Spectral Clustering.

Для подбора гиперпараметров используется библиотека **Optuna**.
Качество моделей оценивается по совокупности метрик, связанных с компактностью групп, связанностью территорий, взаимодополняемостью инфраструктуры и пространственной близостью.

## Назначение приложения

Данная версия приложения используется для выбора наиболее подходящей модели кластеризации на данных Нижегородской области.

По результатам экспериментов в качестве основной модели для дальнейшего применения была выбрана **Spectral Clustering**. На её основе подготовлена универсальная версия приложения для применения к данным другого региона.

## Состав репозитория

```text
app.py                  # Streamlit-приложение
model_pipeline.py       # расчётная часть модели
requirements.txt        # зависимости проекта
population_NN.xlsx      # данные о численности населения
results_NN.xlsx         # данные об объектах инфраструктуры
distances_matrix_NN.xlsx # матрица расстояний между территориями
```

## Запуск приложения

Для локального запуска необходимо установить зависимости:

```bash
pip install -r requirements.txt
```

Затем запустить приложение:

```bash
streamlit run app.py
```
